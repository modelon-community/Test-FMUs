About
-----------

This repository contains FMUs used in testing of PyFMI.

These are not meant for distribution or cross-tool verification at this point.

How to generate FMUs in Modelon Impact
-----------
1. Open [Impact](https://impact.modelon.cloud/).
2. Go to JupyterLab.
3. Clone this repository.
4. Open terminal and run 'python compile.py'.
5. Download the generated 'FMUs.zip'.
6. Create release and add 'FMUs.zip' as asset.
